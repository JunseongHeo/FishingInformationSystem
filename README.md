
# Fishing Life

### :ballot_box_with_check: 프로젝트 개요
낚시꾼들이 필요한 정보를 제공하는 낚시정보 종합 웹사이트 제작

### :date: 프로젝트 기간
7월 4일(월) ~ 7월 28일(목), 4주 동안 진행

### :blush: 참여자 
조진원, 주재영, 이건효, 이정연, 허준성
